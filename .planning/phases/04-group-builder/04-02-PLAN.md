    Named export `ConflictOverlay`.

    Props:
    ```typescript
    type ConflictOverlayProps = {
      conflicts: Array<{ idA: string; idB: string; status: 'conflict' | 'unknown' }>
      cardRefs: React.RefObject<Map<string, HTMLElement>>

      onConflictClick: (idA: string, idB: string) => void
    }
    ```

    Implementation:
    - Uses `useLayoutEffect` to compute line positions from DOM rects
    - Dependencies for useLayoutEffect: `conflicts`, `cardRefs` (and a deps-trigger from parent — see below)
    - Reads `containerRef.current.getBoundingClientRect()` as the origin
    - For each conflict where `status === 'conflict'` (per D-04: only conflict, NOT unknown):
      - Look up `cardRefs.current.get(c.idA)` and `cardRefs.current.get(c.idB)`
      - If both exist, compute center points relative to container
      - Store as `{ x1, y1, x2, y2, idA, idB }` in local state
    - Render absolute-positioned SVG:
      ```tsx
      <svg className='absolute inset-0 w-full h-full pointer-events-none' style={{ zIndex: 10 }}>
        {lines.map((l) => (
          <line
            key={`${l.idA}-${l.idB}`}
            x1={l.x1} y1={l.y1} x2={l.x2} y2={l.y2}
            stroke='#ef4444'
            strokeWidth={1.5}
            opacity={0.7}
            style={{ pointerEvents: 'all', cursor: 'pointer' }}
            onClick={() => onConflictClick(l.idA, l.idB)}
          />
        ))}
      </svg>
      ```

    Important: The useLayoutEffect must include `conflicts.length` and a stringified version of the dog IDs in the group (passed as a prop or derived) so it recomputes when dogs are added/removed. Use `JSON.stringify(conflicts)` as a dependency for simplicity.
